# Brand logo placement

Put your final brand logo here as `logo.svg`.

- Path: `public/brand/logo.svg`
- Format: SVG preferred (vector)
- Once added, run `npm run build` to regenerate:
  - Favicons: `/favicon-16x16.png`, `/favicon-32x32.png`
  - Apple Touch Icon: `/apple-touch-icon.png`
  - PWA icon: `/icon-512.png`
  - Open Graph images: `/og-image.png`, `/og-home.png`, `/og-fleet.png`, `/og-support.png`

If this file is missing, a fallback monogram will be used automatically.
